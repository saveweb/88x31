## 这是什么网站

一些旧网站上有者像徽章、按钮一样的矩形横幅，通常是五颜六色，甚至带有动画效果的 GIF 或 PNG 格式的图片。

它有这些名字:

1.  The 88x31 buttons
2.  Web banner
3.  <ruby>Badge<rp>(</rp><rt>徽章</rt><rp>)</rp></ruby>
4.  antipixel（特指 80 × 15 的大小）

    Steal These Buttons (STB)

此网站收集关于此图片的一切资料，并不仅局限于 88 × 31 的大小。
